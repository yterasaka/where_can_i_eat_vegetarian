# Features

- Select a city to display restaurants (Tokyo, Yokohama, Nagoya, Kyoto, Osaka)
- User authentication functions (register, login, logout, reset password)
- Favorite function (for registered users)

# Technology used

- Next.js
- JavaScript
- Google Maps API
- Yelp API
- Strapi
- Vercel
- Railway
